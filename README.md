# 10 ミニゲーム3本セット（箱）

このアプリは、3つのミニゲーム（じゃんけん、数字当て、タイマー）を1つのページにまとめたものです。タブまたはセクションごとに個別のゲームを楽しむことができます。

## ゲーム一覧

1. **じゃんけん** — ボタンを押すと自動的に自分と相手の手を決めて、勝敗を判定します。
2. **数字当てゲーム** — コンピュータが1〜100の中から選んだ数字を予想します。ヒント（高い/低い）が表示され、当てるまで続けられます。新しいゲームの開始も可能です。
3. **タイマー** — 任意の秒数を設定してスタートする簡易タイマーです。カウントダウンが終了すると終了メッセージが表示されます。

## 使い方

1. 以下のコマンドでアプリを起動します。

    ```bash
    streamlit run app.py
    ```

2. アプリが起動したら、各ゲームのタブに切り替えることで遊びたいゲームを選べます。

3. それぞれのゲームの指示に従い、ボタンや入力欄を利用してゲームを進めてください。

このアプリは複数機能を1画面にまとめる方法や、`st.tabs` を利用したページ分割の例として活用できます。
